# z80
Some z80 programs that I've made


addmany_arr8_8.s:

Takes two arguments: UBYTE,UBYTE*
where UBYTE is the size of the array
Returns an 8 bit result given an array of BYTE


addmany_arr16_16.s:
  
Takes two arguments: UBYTE,UBYTE*
where UBYTE is the size of the array 
Returns a 16 bit result given an array of INT16
